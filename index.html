웹 기술의 발전

몇년이 지난 후 사람들은 인터넷에서 더 즐거운 것들을 하길 원했고 이러한 욕망은 멀티미디어 웹으로 반영되었다.

플래시와 엑티브X가 사용되었고, 웹 사이트 구조는 점점 더 복잡해졌다.

이에 따라 웹 개발자에 많은 종류가 생겼다.

그리고 이 시기에 웹사이트들은 모두 정적인 웹사이트였다.

현재의 웹사이트

동적인 요소가 점점 더 중요시 되면서 HTML,CSS보다 JavaScript 위주로 코드가 작성되기 시작했다.

UI / UX를 담당하는 부분인 프론트엔드,
Data와 통신하고 처리하는 부분인 백엔드의 경계가 명확하게 나뉘어지게 되고,

엄청난 동시 접속자 수와 방대한 Data의 양때문에 웹사이트가 매우 복잡하고 정교해진다.

What is Web Developer

Front-end

비지니스 로직과 사용자 영역의 개발을 담당

Back-end

데이터 베이스와 API 서버 개발을 담당
